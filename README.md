# 버스 챗봇
광주광역시 버스 정보 카카오 i 오픈빌더 챗봇.

## 소개
  - 실시간 버스 도착정보 검색
광주광역시의 버스 정류장 이름을 입력하면 해당 정류장에 도착하는 버스에 대한 정보를 알려줍니다.

## 개발환경
- 개발툴 : Atom
- 언어 : Node.js
- DB : Firebase
- 배포 : Amazone EC2
- API : 광주광역시 BIS 정류소 정보 , 광주광역시 BIS 도착정보
```
npm install
node app.js
```
